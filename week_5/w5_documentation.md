# Week 5 Documentation
## Problem Statement
Create a Python program to calculate a customer's café bill automatically.
## Inputs
- Customer name
- Coffee quantity
- Tea quantity
- Sandwich quantity
## Outputs
- Customer name
- Coffee quantity
- Tea quantity
- Sandwich quantity
- Total bill
## Process Flow
1. Enter customer name.
2. Enter quantities.
3. Calculate total.
4. Print receipt.
## Constraints
- Quantities must be positive.
- Coffee = RM8.50
- Tea = RM6.00
- Sandwich = RM12.00

START
Input customer name
Input coffee quantity
Input tea quantity
Input sandwich quantity
